Gym Bro - Khởi chạy không cần Node.js

Lựa chọn phương pháp chạy ứng dụng

Lưu ý: Phương pháp này sử dụng CDN thay vì cài đặt thư viện địa phương. Một số tính năng có thể bị giới hạn.

1. Sử dụng Live Server trong VS Code

Nếu bạn đang sử dụng Visual Studio Code:

  1. Cài đặt extension Live Server
  2. Click chuột phải vào file index.html
  3. Chọn "Open with Live Server"

2. Sử dụng trình duyệt trực tiếp

Mở trực tiếp file index.html với trình duyệt của bạn:

  1. Mở File Explorer/Finder
  2. Tìm đến thư mục dự án
  3. Double-click vào file index.html

Hoặc bạn có thể kéo file index.html vào cửa sổ trình duyệt.

3. Sử dụng Python SimpleHTTPServer

Nếu bạn đã cài đặt Python:

  1. Mở terminal/command prompt
  2. Di chuyển đến thư mục dự án của bạn
  3. Nhập lệnh sau:
Python 3: python -m http.server
Python 2: python -m SimpleHTTPServer

Sau đó truy cập http://localhost:8000

Kiểm tra nếu gặp lỗi

Nếu trang web không hoạt động đúng cách, bạn có thể kiểm tra:

  1. Mở Developer Console trong trình duyệt (F12) và xem lỗi nào đang xuất hiện
  2. Kiểm tra xem tất cả các file .js và .css có tồn tại trong cùng thư mục không
  3. Kiểm tra xem thư mục images đã được tạo và chứa file logo.png chưa
  4. Thử mở file test.html để kiểm tra các thư viện
