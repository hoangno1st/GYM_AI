Tạo Logo cho Gym Bro

Dùng công cụ này để tạo một logo đơn giản nếu bạn không có file logo.png

Lưu ý: Chức năng này chỉ tạo logo tạm thời. Bạn nên thay thế bằng logo chính thức sau.

Hướng dẫn sử dụng:

  1. Điều chỉnh các tùy chọn theo ý muốn
  2. Nhấn "Cập nhật" để xem kết quả
  3. Nhấn "Tải logo" để tải file logo.png
  4. Đặt file logo.png vào thư mục "images"
